## Setup Instructions  

To run the website locally, follow these steps:  

#### Clone the Repository  
```sh
git clone https://github.com/vickyy234/spark.git
```

#### Navigate to the Project Directory
```sh
cd spark
```

#### Install Dependencies
```sh
npm install
```

#### *Or*
```sh
npm i
```

#### Start the Development Server
```sh
npm run dev
```

#### Open in Browser
```sh
http://localhost:5173
```
